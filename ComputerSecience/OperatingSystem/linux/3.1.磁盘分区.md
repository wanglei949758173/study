# 磁盘分区

## 分区基础知识

### 分区的方式

* mbr分区
  1. 最多支持四个主分区
  2. 系统只能安装在主分区
  3. 扩展分区要占一个主分区
  4. MBR最大支持2TB，但拥有最好的兼容性
* gpt分区
  1. 支持无限多个主分区（但操作系统可能限制，比如windows下最多128个分区）
  2. 最大支持18EB的大容量（1EB=1024PB，1PB=1024TB）
  3. windows7 64位后支持gpt
