# be动词的基本用法
be的意思是“是”

be最基本的形式有三种：am,is,are

## am
**am：只用于第一人称单数I作主语时**。
```
I am your Father.
```

## is
**is:只用于第三人称单数he,she,it作主语时**
```
He is my friend.
She is my sister.
It is my dog.
```

## are
**are:用于第二人称单数以及所有人称的复数作主语时**

### 第二人称单数/复数
```
You are beautiful.
```

### 第一人称复数
```
We are a family.
```

### 第三人称复数
```
They are my friends.
```

## am,is,are缩写
am经常在主语后缩写为'm,is经常在主语后缩写为's，are经常在主语后缩写为'are。
```
I'm a teacher.
He's my friend.
She's my sister.
You're beautiful.
```
