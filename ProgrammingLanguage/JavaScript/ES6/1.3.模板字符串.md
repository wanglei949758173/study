# 模板字符串

模板字符串（template string）是增强版的字符串，用反引号（`）标识，具有以下特点：

* 字符串中可以出现换行
* 可以使用`${xxx}`形式输出变量

```javascript
// 定义字符串
let str = `<ul>
            <li>沈腾</li>

            <li>玛丽</li>

            <li>魏翔</li>

            <li>艾伦</li>
           </ul>`;

// 变量拼接
let star = '王宁';

let result = `${star}在前几年离开了开心麻花`;
```

**使用场景：当遇到字符串与变量拼接的情况下使用模板字符串。**
